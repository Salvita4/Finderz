# Finderz

<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/11482b78-6258-4010-959b-793df0acac14" />

Finderz es una plataforma tecnológica para eventos masivos que combina dispositivos IoT, comunicación LoRa y una aplicación móvil para facilitar el encuentro entre personas y la navegación dentro de un predio sin necesidad de conexión a internet.

A través de una brújula inteligente y un mapa interactivo, los usuarios pueden localizar amigos, acceder a puntos de interés, recibir promociones dinámicas y encontrar rutas seguras dentro del evento. Además, Finderz incorpora herramientas de análisis y gestión para organizadores, permitiéndoles comprender el comportamiento de los asistentes y optimizar la experiencia general del evento.

## Componentes

- 🧭 Brújula Inteligente (IoT + LoRa)
- 📱 Aplicación Mobile
- 🌐 Backend y APIs
- 📊 Back Office para Organizadores

## Objetivo

Transformar los desencuentros en encuentros, convirtiendo la orientación dentro de eventos masivos en una experiencia interactiva, segura y divertida.
